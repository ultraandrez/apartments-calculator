## Калькулятор ипотеки 
Запуск контейнера 
````
docker-compose up --build 
````
Переходим в директорию проекта
````
docker-compose exec web bash 
````
Генерируем ключ и устанавливаем миграции
````
php artisan key:generate
php artisan migrate
````
